# PDF Maestro 📄👑

*Read this in other languages: [English](README_en.md), [Español](README.md).*

**PDF Maestro** is a local, private, and unlimited desktop application designed to manage and edit your PDF files quickly and securely, without relying on internet connections or external servers.

---

## 🚀 Key Features

*   **🔒 100% Private and Local:** Your documents never leave your computer. All processing is done locally to ensure maximum security and privacy.
*   **➕ Merge PDF:** Combine multiple PDF files into a single document in seconds.
*   **✂️ Split PDF:** Separate pages from a PDF file or extract specific ranges.
*   **🔄 Convert:** Easily convert images to PDF and vice versa.
*   **📉 Compress PDF:** Reduce the size of your PDF files without noticeable quality loss.
*   **💧 Watermark:** Add custom watermarks to protect your documents.
*   **🛡️ PDF Security:** Protect your files with passwords or remove reading restrictions.
*   **🔲 Pro QR Generator:** Create custom high-resolution (1200x1200px) QR codes with your logo in the center, ideal for printing and commercial use.

---

## 🛠️ Technologies Used

*   **Electron:** Framework for creating native desktop applications using HTML, CSS, and JavaScript.
*   **pdf-lib:** Robust library for PDF document manipulation and editing.
*   **qr-code-styling:** Professional tool for QR code generation and design.

## 💻 Installation and Development

If you want to run the project in your local environment or contribute to the development:

### Prerequisites
You must have [Node.js](https://nodejs.org/) installed (version 18 or higher recommended).

### Getting Started:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/LTCBlackjack/PDF-Maestro.git
    cd pdf-maestro
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the application in development mode:**
    ```bash
    npm start
    ```

4.  **Build for production (Create the .exe installer for Windows):**
    ```bash
    npm run build
    ```
    *The executable files ready for distribution will be automatically generated in the `dist/` folder.*
